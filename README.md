# Online_Billing_Application
A modern and responsive Online Billing System built using HTML, CSS, JavaScript, Node.js, and MySQL. This project allows users to register, login, create bills, manage products, and generate invoices with a clean UI.

## 🚀 Features

* 🔐 User Authentication (Register & Login)
* 🧾 Create and Manage Bills
* 📦 Add Products (Manual Entry / Barcode Mode UI)
* 💰 Automatic Price Calculation
* 🏷️ Discount System
* 🧮 Subtotal & Net Total Calculation
* 💾 Save Bills (Database Ready)
* 🖨️ Print Invoice
* 🎨 Modern Glassmorphism UI with Animated Background
* 📱 Fully Responsive Design

## 📂 Project Structure

```
online-billing-system/
│
├── index.html          # Frontend UI
├── server.js           # Backend Server
├── schema.sql          # Database Schema
├── package.json        # Node Dependencies
└── README.md           # Project Documentation
```


### 2️⃣ Install Dependencies

```
npm install
```

### 3️⃣ Setup Database

* Open MySQL / phpMyAdmin
* Import `schema.sql`

### 4️⃣ Run Server

```
node server.js
```

### 5️⃣ Open Application

Open `index.html` in your browser
(or use Live Server)
